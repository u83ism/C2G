# 概要
Silent Stormは2003年に Nival Interactive から発売された、第2次世界大戦をテーマにしたターン制タクティカルRPGです。（~~パンツァークラインとビーム兵器は見なかったことにする~~）

ゲーム性としては[XCOM](https://store.steampowered.com/app/200510/XCOM_Enemy_Unknown/?l=japanese)の舞台を第2次世界大戦にして建物破壊ギミックを盛りまくった感じです。あとやったことないですけどJagged Alliance系にも近いんですかね？よく類似ゲームとしてお互い紹介されます。

この手のゲームはハマる人にはドハマりして、小さいながらも熱狂的なファンコミュニティがある一方で、（元からSRPGって開発難度高いと言われますが）その中でも開発難度が非常に高く頓挫するケースが非常に多いです。もはや発売されるだけでお祝いレベル、まともに遊べるレベルは奇跡という感じです。そのためJA2にせよX-COMにせよ延々と古いゲームを改造しながら遊び続けてる、みたいな感じですけど、その中でもS2は上澄みな方と言われています。自分はSFがいまいち好きじゃないのと、建物破壊システムが気に入ってS2にハマったって感じですね。（~~パンツァークラインとビーム兵器は見なかったことにする~~） [redditで書かれていたレビュー](https://www.reddit.com/r/patientgamers/comments/kvqsyn/i_fell_in_love_with_silent_storm_again_after_15/?tl=ja) にはかなり同意。

現在は[Steam](https://store.steampowered.com/app/254960/Silent_Storm_Gold_Edition/?l=japanese)や[GOG](https://www.gog.com/en/game/silent_storm_gold)で拡張版のSilent Storm Sentinels入りのバージョンがGold Editionとして販売されていますが、日本語は未対応です。一方、今は亡きメディアクエスト（00年代前半に洋ゲーの翻訳版を精力的に販売してその界隈では有名だった）から日本語版がパッケージで出てたので、オークション等で探せば入手は可能です。ただSilent Storm Sentinelsが入ってないのと、下記バイナリ書き換え対応が必要で、注意が必要です。

# 【日本語版限定】「サポートされてないビデオカードです」問題
S2の日本語版はベースとなったバージョンが古く（v1.2）、インストール後起動しようとすると「サポートされてないビデオカードです。」と表示されます。これに対しては非公式パッチがあるが、日本語版のバイナリには対応してないので直接着替えが必要です。

バイナリの書き換えには専用のエディタ（ここでは日本語版もある[HxD](https://mh-nexus.de/en/hxd/)を使うものとする）を使います。昔はPCゲームの改造と言えばProgram Files内のファイルを直接書き換えるものが多かったが、今はセキュリティが強化されているのでHxDは**管理者権限で起動する必要があります。**

`Program Files (x86)\JoWooD\Silent Storm\game.exe`の`3B C7 74 2E`と並んでいる箇所の**74**を**EB**に書き換える必要がありますが、日本語版はアドレスが若干変わっており、`Offset(H)=00009A90/0E` になります。

# 不具合/不満点
有名どころだと

- 控えメンバーの自動成長機能が強すぎて終盤はレギュラーメンバーよりも強くなってしまう
- メディカルとエンジニアのスキルの要求ポイントが高すぎて後半上げられなくなる
- グレネードの投擲距離が短すぎる
    - これはわざとかも。MODを入れると最悪視認できてない敵が一方的に遠投してきてえらいことになるし……逆もしかりですが

# OSS化
実は2026年初頭にソースコードが[Githubに公開](https://github.com/nival/Silent-Storm)されて、非商用目的限定でOSS化されてます。
ただ[当時Git管理してなかったらしく、古い上に欠けてるデータなどもあり現行の最新版どころか初回販売版ですら再現できない](https://github.com/nival/Silent-Storm/issues/3)ようですが、プログラマーの人はAI片手に解析してみるのも面白いと思います。
